    # NiivueOptions

    Niivue exposes many properties. It's always good to call `updateGLVolume` after altering one of these settings.
    Type:
    • Object
    Properties
    Name Type Attributes Default Description
    options.textHeight number <optional>
    		 0.06 the text height for orientation labels (0 to 1). Zero for no text labels
    options.colorbarHeight number <optional>
    		 0.05 size of colorbar. 0 for no colorbars, fraction of Nifti j dimension
    options.colorbarMargin number <optional>
    		 0.05 padding around colorbar when displayed
    options.crosshairWidth number <optional>
    		 1 crosshair size. Zero for no crosshair
    options.rulerWidth number <optional>
    		 4 ruler size. Zero (or isRuler is false) for no ruler
    options.backColor array <optional>
    		 [0,0,0,1] the background color. RGBA values from 0 to 1. Default is black
    options.crosshairColor array <optional>
    		 [1,0,0,1] the crosshair color. RGBA values from 0 to 1. Default is red
    options.fontColor array <optional>
    		 [0.5,0.5,0.5,1] the font color. RGBA values from 0 to 1. Default is gray
    options.selectionBoxColor array <optional>
    		 [1,1,1,0.5] the selection box color when the intensty selection box is shown (right click and drag). RGBA values from 0 to 1. Default is transparent white
    options.clipPlaneColor array <optional>
    		 [1,1,1,0.5] the color of the visible clip plane. RGBA values from 0 to 1. Default is white
    options.rulerColor array <optional>
    		 [1, 0, 0, 0.8] the color of the ruler. RGBA values from 0 to 1. Default is translucent red
    options.show3Dcrosshair boolean <optional>
    		 false true/false whether crosshairs are shown on 3D rendering
    options.trustCalMinMax boolean <optional>
    		 true true/false whether to trust the nifti header values for cal_min and cal_max. Trusting them results in faster loading because we skip computing these values from the data
    options.clipPlaneHotKey string <optional>
    		 "KeyC" the keyboard key used to cycle through clip plane orientations. The default is "c"
    options.viewModeHotKey string <optional>
    		 "KeyV" the keyboard key used to cycle through view modes. The default is "v"
    options.keyDebounceTime number <optional>
    		 50 the keyUp debounce time in milliseconds. The default is 50 ms. You must wait this long before a new hot-key keystroke will be registered by the event listener
    options.doubleTouchTimeout number <optional>
    		 500 the maximum time in milliseconds for a double touch to be detected. The default is 500 ms
    options.longTouchTimeout number <optional>
    		 1000 the minimum time in milliseconds for a touch to count as long touch. The default is 1000 ms
    options.isRadiologicalConvention boolean <optional>
    		 false whether or not to use radiological convention in the display
    options.logging boolean <optional>
    		 false turn on logging or not (true/false)
    options.loadingText string <optional>
    		 "waiting on images..." the loading text to display when there is a blank canvas and no images
    options.dragAndDropEnabled boolean <optional>
    		 true whether or not to allow file and url drag and drop on the canvas
    options.isNearestInterpolation boolean <optional>
    		 false whether nearest neighbor interpolation is used, else linear interpolation
    options.isAtlasOutline boolean <optional>
    		 false whether atlas maps are only visible at the boundary of regions
    options.isRuler boolean <optional>
    		 false whether a 10cm ruler is displayed
    options.isColorbar boolean <optional>
    		 false whether colorbar(s) are shown illustrating values for color maps
    options.isOrientCube boolean <optional>
    		 false whether orientation cube is shown for 3D renderings
    options.multiplanarPadPixels number <optional>
    		 0 spacing between tiles of a multiplanar view
    options.multiplanarForceRender boolean <optional>
    		 false always show rendering in multiplanar view
    options.meshThicknessOn2D number <optional>
    		 Infinity 2D slice views can show meshes within this range. Meshes only visible in sliceMM (world space) mode
    options.dragMode DRAG_MODE <optional>
    		 contrast behavior for dragging (none, contrast, measurement, pan)
    options.isDepthPickMesh boolean <optional>
    		 false when both voxel-based image and mesh is loaded, will depth picking be able to detect mesh or only voxels
    options.isCornerOrientationText boolean <optional>
    		 false should slice text be shown in the upper right corner instead of the center of left and top axes?
    options.sagittalNoseLeft boolean <optional>
    		 false should 2D sagittal slices show the anterior direction toward the left or right?
    options.isSliceMM boolean <optional>
    		 false are images aligned to voxel space (false) or world space (true)
    options.isHighResolutionCapable boolean <optional>
    		 true demand that high-dot-per-inch displays use native voxel size
    options.drawingEnabled boolean <optional>
    		 false allow user to create and edit voxel-based drawings
    options.penValue number <optional>
    		 Infinity color of drawing when user drags mouse (if drawingEnabled)
    options.isFilledPen boolean <optional>
    		 false create filled drawings when user drags mouse (if drawingEnabled)
    options.maxDrawUndoBitmaps number <optional>
    		 8 number of possible undo steps (if drawingEnabled)
    options.thumbnail string <optional>
    		 "" optional 2D png bitmap that can be rapidly loaded to defer slow loading of 3D image

    name string <optional>
    		 "" a name for this image. Default is an empty string

    		 NaN minimum intensity for color brightness/contrast
    cal_max number <optional>
    		 NaN maximum intensity for color brightness/contrast
